# Employees_syetem
A Python-based Employee data management system utilizing Object-Oriented Programming (OOP) to perform CRUD operations. Includes functionality to add new employees, filter/delete records by age, update salary information by name, and persist data in an Excel format using Pandas.

Add New Employees: Easily input employee details (Name, Age, Salary) to the system.

Data Visualization: View a list of all currently registered employees.

Delete by Age: Efficiently remove employees from the database based on a specific age range.

Export to Excel: Automatically save the current employee database to an .xlsx file using the Pandas library, ensuring data persistence.

Interactive Menu: A user-friendly console interface to navigate through the system's functionalities.
